# AI-Timetable-Scheduler
EduChronos is a full-stack, AI-powered academic timetable scheduling system designed to automatically generate conflict-free weekly schedules for educational institutions. Built with a Python/Flask backend and a lightweight HTML/JS/CSS frontend, it eliminates the tedious manual process of timetable creation.


---

##  Getting Started

### Prerequisites

```bash
pip install flask numpy pandas
```

### Run the App

```bash
python app.py
```

Then open your browser at **`http://localhost:5000`**

### Quick Start with Sample Data

1. Open the app in your browser
2. Upload the `*_sample.csv` files (teachers, rooms, subjects, classes)
3. Configure days per week and periods per day
4. Click **Generate Timetable**
5. View, analyze, and export your conflict-free schedule!
